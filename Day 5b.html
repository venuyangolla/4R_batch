Image Tag

The "img" tag is used to embed an image in an html page. images are not techinaclly inserted in the web pages

The tag requires two attributes:
src- specifies the path to image
alt-specifies the alternate text for the image, if the image for some reason cannot dispalyed

Figure Tag:

The Figure tag is used to insert self contained content like diagrams etc. It can be placed at any position in the document and related to main flow.

Figcaption Tag:

The fig caption tag in HTML is used to caption for the fogure element. It can be place at the first and last one of the element.

Link Tag:

The link tag can be written as and no end tags, the rel attributes specifies the relationship b/w the document containing the hyperlink and destination source.
