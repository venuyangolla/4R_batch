List

It has 3 types.
1. Ordered List.
2. Unordered List
3. Discription List

Ordered List: It should be written in the "ol" tag. It should be written in Order by used in the number or alphabets.
Ex:
  1. raju
  2. venkat
  3. tree
Unordered List It has written in the "ul" tag. . It does not contain any number it represent in the dots etc.
Ex:
  • movies
  • games
  • outing
Discription List It is written in "dl" tag. "dt" is defines that term discription. "dd" is defines the discription definition.
Ex:
journey
we are on the way
